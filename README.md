# k3s-vps-homelab
k3s homelab setup, based on https://k3s.rocks/

It is hosted in [Contabo](https://contabo.com/en/vps-c/).   
VPS spec:
* 4 vCPU Cores
* 6 GB RAM
* 400 GB SSD

Pricing: U$ 5.50/month per VPS.
